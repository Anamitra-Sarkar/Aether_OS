# AetherOS Artwork

This directory contains placeholder artwork for AetherOS.

## Files

- `logo.svg` - Main AetherOS logo (128x128)
- `wallpaper-4k.svg` - Default wallpaper (3840x2160)
- `icons/` - Custom application icons

## Design Guidelines

### Colors

- Primary Accent: `#6C8CFF` (Aether Blue)
- Secondary Accent: `#7AE7C7` (Soft Mint)
- Background: `#0F1720` (Dark Blue-Gray)
- Surface: `#101317` (Darker)

### Logo Usage

The logo should be used on:
- Boot splash screen
- Login screen
- About dialogs
- Application launcher

### Wallpaper

The default wallpaper features:
- Subtle gradient background
- Floating particles
- Abstract geometry
- Low contrast to not distract from desktop content

## Creating Derivative Works

When creating new artwork:
1. Follow the color palette in `docs/design_tokens.md`
2. Use rounded corners (10-12px radius)
3. Maintain consistency with existing elements
4. Export in appropriate formats (SVG for scalable, PNG for raster)
