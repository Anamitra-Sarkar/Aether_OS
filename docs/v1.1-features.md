# AetherOS v1.1 Features Guide

This guide explains the new quality-of-life features introduced in AetherOS v1.1.

---

## Focus Mode / Do Not Disturb

Focus Mode silences all notifications, helping you concentrate on your work.

### How to Use

**Via Command Line:**
```bash
# Toggle Focus Mode on/off
/usr/share/aetheros/scripts/aether-focus-mode.sh toggle

# Turn on Focus Mode
/usr/share/aetheros/scripts/aether-focus-mode.sh on

# Turn off Focus Mode
/usr/share/aetheros/scripts/aether-focus-mode.sh off

# Check current status
/usr/share/aetheros/scripts/aether-focus-mode.sh status
```

**Via Application Launcher:**
- Search for "Toggle Focus Mode" in the application menu
- Click to toggle Focus Mode on/off

**Via Keyboard Shortcut (optional):**
You can set a custom keyboard shortcut in:
- System Settings → Shortcuts → Custom Shortcuts
- Command: `/usr/share/aetheros/scripts/aether-focus-mode.sh toggle`

## Auto Light/Dark Theme Schedule

Automatically switches between light and dark themes based on time of day.

### Default Schedule
- **Light Theme**: 7:00 AM - 7:00 PM
- **Dark Theme**: 7:00 PM - 7:00 AM

### How to Use

**Enable Auto Schedule:**
```bash
/usr/share/aetheros/scripts/aether-theme-scheduler.sh enable
```

**Disable Auto Schedule:**
```bash
/usr/share/aetheros/scripts/aether-theme-scheduler.sh disable
```

**Check Status:**
```bash
/usr/share/aetheros/scripts/aether-theme-scheduler.sh status
```

**Apply Theme Immediately (based on current time):**
```bash
/usr/share/aetheros/scripts/aether-theme-scheduler.sh apply
```

### Configuration

The schedule can be customized by editing:
```
~/.config/aetheros/theme-scheduler.conf
```

Change `LIGHT_START_HOUR` and `LIGHT_END_HOUR` to your preferred times (in 24-hour format).

## System Sound Theme

AetherOS v1.1 includes a minimal, clean sound theme for login, notifications, and alerts.

### How to Use

**Enable Sounds:**
```bash
/usr/share/aetheros/scripts/aether-sounds.sh enable
```

**Disable Sounds:**
```bash
/usr/share/aetheros/scripts/aether-sounds.sh disable
```

**Toggle Sounds:**
```bash
/usr/share/aetheros/scripts/aether-sounds.sh toggle
```

**Test Sound:**
```bash
/usr/share/aetheros/scripts/aether-sounds.sh test
```

**Via Application Launcher:**
- Search for "Toggle System Sounds" in the application menu
- Click to toggle sounds on/off

### Sound Theme Details

AetherOS uses the KDE Ocean sound theme, which provides:
- Subtle login sound
- Notification chime
- Error/alert sounds
- All sounds are minimal and non-intrusive

## New Wallpapers

AetherOS v1.1 adds two new wallpapers:

1. **Aether Waves** - Flowing waves with gradient colors
2. **Aether Minimal** - Minimal geometric design with central focus

### How to Change Wallpaper

1. Right-click on desktop → Configure Desktop and Wallpaper
2. Click "Add Image..." button
3. Navigate to `/usr/share/backgrounds/aetheros/`
4. Select your preferred wallpaper

Or via System Settings:
- System Settings → Appearance → Wallpaper

## Tips

### Focus Mode Best Practices
- Enable Focus Mode during meetings, coding sessions, or focused work
- Set a keyboard shortcut for quick access
- Use with "Do Not Disturb" in communication apps for maximum focus

### Theme Scheduling Tips
- The scheduler checks every hour
- Disable auto-schedule if you prefer manual theme control
- You can manually apply themes using `/usr/share/aetheros/scripts/apply-theme.sh light` or `dark`

### Sound Configuration
- Sounds are enabled by default
- Disable them if you prefer a completely silent system
- Test sounds after enabling to ensure audio is working correctly

## Troubleshooting

### Focus Mode Not Working
- Ensure KDE Plasma is running
- Check notification settings in System Settings
- Restart Plasma: `killall plasmashell && plasmashell &`

### Theme Scheduler Not Switching
- Check timer status: `systemctl --user status aether-theme-scheduler.timer`
- Restart timer: `systemctl --user restart aether-theme-scheduler.timer`
- Apply manually: `/usr/share/aetheros/scripts/aether-theme-scheduler.sh apply`

### No Sound Playback
- Check audio output in System Settings → Audio
- Test with: `paplay /usr/share/sounds/freedesktop/stereo/message.oga`
- Verify sound theme: `kreadconfig5 --file kdeglobals --group Sounds --key Theme`
