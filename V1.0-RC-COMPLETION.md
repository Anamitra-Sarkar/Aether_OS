## CI/CD Pipeline

### Build Job
- Installs dependencies
- Builds ISO (full or minimal)
- Uploads ISO artifact
- Uploads SHA256 checksum
- Uploads build log

### Test Job
- Downloads ISO artifact
- Runs QEMU boot test (180s timeout)
- Captures screenshots
- Runs UI sanity checks
- Verifies required assets
- Uploads test results
- Uploads screenshots

### Lint Job
- Runs shellcheck on all scripts
- Checks script permissions
- Validates error handling

### CodeQL Job
- Scans C++ and Python code
- Runs security-and-quality queries
- Reports findings

### Release Job (on tag)
- Creates GitHub release
- Attaches ISO and SHA256
- Includes screenshots
- Uses release notes template
