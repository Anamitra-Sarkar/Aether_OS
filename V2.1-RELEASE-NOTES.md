## üîí Security Features

### AetherShield - Per-App Sandbox Control (Phase 1)

Your first line of defense with granular app permissions:

```bash
# List managed applications
aethershieldctl list

# View policy for an app
aethershieldctl show firefox

# Apply policy restrictions
aethershieldctl apply firefox

# Check current status
aethershieldctl status firefox
```

**Features**:
- JSON-based policy manifests
- Control network, camera, microphone, filesystem access
- Integration with AppArmor and Flatpak
- Opt-in enforcement (no apps broken by default)
- Phase 1: Policy awareness + partial enforcement

### Secure Session Mode

Lock down your system for sensitive tasks:

```bash
# Enable secure mode
aether-secure-session.sh start

# Check status
aether-secure-session.sh status

# Return to normal
aether-secure-session.sh stop
```

**When Active**:
- ‚úì Strict firewall rules (deny incoming)
- ‚úì SSH server disabled
- ‚úì Network services stopped (Avahi, Samba)
- ‚úì USB automount disabled
- ‚úì Visual notification indicator
- ‚úì All changes reversible

**Perfect For**:
- Online banking and financial transactions
- Exam portals and online tests
- Confidential work and sensitive documents

## üå°Ô∏è Thermal Intelligence

### Thermal Watch - Heat-Aware Visuals

Your system now reacts intelligently to temperature:

```bash
# Check current temperature and state
aether-thermal-watch.sh check

# Run monitoring daemon
aether-thermal-watch.sh monitor

# Enable as systemd service
systemctl --user enable --now aether-thermal.service
```

**How It Works**:
- **Cool** (<60¬∞C): Full visual effects
- **Warm** (60-75¬∞C): Reduced effects
- **Hot** (>75¬∞C): Performance mode activated

**Safety Features**:
- Minimum 60s between changes (no thrashing)
- Respects user profile overrides
- Automatic restoration when system cools
- Comprehensive logging

## ‚ôø Accessibility Improvements

### Accessibility Manager

Make AetherOS comfortable for everyone:

```bash
# Disable all animations
aether-accessibility.sh reduce-motion on

# Enable high contrast
aether-accessibility.sh high-contrast on

# Check current settings
aether-accessibility.sh status
```

**Reduced Motion Mode**:
- Disables all animations and transitions
- Removes blur effects
- Helpful for vestibular disorders
- Perfect for static UI preference

**High Contrast Mode**:
- Increases text readability
- Reduces transparency
- Better for low vision users
- Improved color contrast

## üéØ System Requirements

**Unchanged from v2.0**:

- **Minimum**: 4GB RAM, Intel HD 3000 or equivalent GPU
- **Recommended**: 8GB+ RAM, mid-range GPU
- **Storage**: 20GB minimum, 30GB recommended
- **Architecture**: x86_64 (ARM64 experimental)

**New Features Impact**:
- Thermal Watch: ~1-2MB RAM when active
- AetherShield: Minimal overhead (policy checks only)
- Secure Session: No permanent overhead
- Audio Profiles: Instant switching, no background daemon
