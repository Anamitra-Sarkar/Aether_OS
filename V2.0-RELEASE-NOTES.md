## ðŸŽ¨ Part 1: Ultimate Aesthetic Upgrade

### Adaptive Blur System
**Script**: `scripts/aether-adaptive-blur.sh`

Three blur modes automatically selected based on GPU:

| Mode | GPU Tier | Blur Strength | Target Hardware |
|------|----------|---------------|-----------------|
| **High Blur** | Strong | 15 | RTX, RX 6000+, Intel Xe |
| **Frosted Lite** | Mid | 8 | HD 4000-6000, older dedicated |
| **Off** | Weak/VM | 0 | Intel HD 3000-, VMs |

**Features**:
- Automatic GPU detection via `lspci`
- Instant switching between modes
- Manual override available
- Zero GPU cost fallback

**Usage**:
```bash
aether-adaptive-blur.sh auto      # Auto-detect and apply
aether-adaptive-blur.sh high      # Force high mode
aether-adaptive-blur.sh frosted   # Force frosted mode
aether-adaptive-blur.sh off       # Disable blur
aether-adaptive-blur.sh status    # Check current mode
```

### CleanMode Toggle
**Script**: `scripts/aether-cleanmode.sh`

One-click performance optimization that:
- Disables all animations
- Removes blur effects
- Simplifies window shadows
- Reduces compositor overhead
- Disables wobbly windows and effects

**Perfect for**:
- Intel HD 3000 and below
- 4GB RAM systems
- Older laptops (2012 and earlier)
- Virtual machines

**Usage**:
```bash
aether-cleanmode.sh on      # Enable performance mode
aether-cleanmode.sh off     # Restore effects
aether-cleanmode.sh toggle  # Switch on/off
aether-cleanmode.sh status  # Check current state
```

## âš¡ Part 2: Performance Intelligence

### Auto Performance Profiler
**Script**: `scripts/aether-performance-profiler.sh`

Automatically detects hardware and selects optimal profile:

| Profile | RAM | GPU | CPU | Settings |
|---------|-----|-----|-----|----------|
| **MaxMode** | 16GB+ | High | Modern | All effects, max quality |
| **Balanced** | 8-12GB | Mid | Mid-range | Moderate effects |
| **LiteMode** | â‰¤4GB | Low | Old | CleanMode, minimal |

**Detection Logic**:
1. Checks total RAM size
2. Detects CPU generation (Intel/AMD)
3. Identifies GPU tier (high/mid/low)
4. Applies optimal profile

**Usage**:
```bash
aether-performance-profiler.sh auto      # Auto-detect
aether-performance-profiler.sh max       # Force MaxMode
aether-performance-profiler.sh balanced  # Force Balanced
aether-performance-profiler.sh lite      # Force LiteMode
aether-performance-profiler.sh status    # Check profile
```

### Smart Service Manager
**Script**: `scripts/aether-smart-services.sh`

Automatically manages system services based on hardware:

| Service | Trigger | RAM Saved |
|---------|---------|-----------|
| **bluetooth** | Hardware present | ~30MB |
| **cups** | Printer detected | ~50MB |
| **avahi** | Network services needed | ~20MB |
| **baloo** | Large doc collection | ~50MB |

**Benefits**:
- Reduces RAM usage by 50-150MB
- Faster boot times
- Better battery life
- Auto-restarts when hardware connected

**Usage**:
```bash
aether-smart-services.sh check            # Auto-optimize
aether-smart-services.sh status           # Show status
aether-smart-services.sh enable bluetooth # Force enable
```

### Adaptive ZRAM Configuration
**Script**: `opt/enable-zram.sh` (updated)

Smart swap compression with adaptive ratios:

| RAM Size | ZRAM Ratio | Swap Size | Algorithm |
|----------|------------|-----------|-----------|
| â‰¤4GB | 33% | ~1.3GB | LZ4 |
| 4-6GB | 50% | ~2.5GB | LZ4 |
| â‰¥8GB | 75% | ~6GB+ | LZ4 |

**Why LZ4**:
- Fastest compression/decompression
- Lower CPU overhead
- Better for real-time responsiveness
- 2-3x faster than zstd

**Changes from v1.x**:
- Switched from zstd to LZ4
- Reversed ratio logic (more ZRAM on higher RAM)
- Better tuned for different memory sizes

## ðŸ§  Part 3: Intelligent Desktop Behavior

### Focus Mode 2.0
**Script**: `scripts/aether-focus-mode.sh` (enhanced)

**New Features**:

1. **Auto-Activation on Fullscreen**
   - Detects when apps go fullscreen
   - Automatically enables Do Not Disturb
   - Restores when exiting fullscreen
   ```bash
   aether-focus-mode.sh auto-fullscreen on
   aether-focus-mode.sh monitor  # Run in foreground
   ```

2. **Scheduled Mode (Study Mode)**
   - Set active hours (e.g., 9 AM - 5 PM)
   - Automatically enables during schedule
   ```bash
   aether-focus-mode.sh schedule 09:00 17:00
   aether-focus-mode.sh schedule off
   ```

3. **Manual Toggle**
   ```bash
   aether-focus-mode.sh on       # Enable
   aether-focus-mode.sh off      # Disable
   aether-focus-mode.sh toggle   # Switch
   aether-focus-mode.sh status   # Check
   ```

### Smart Notifications
**Script**: `scripts/aether-smart-notifications.sh`

Automatically mutes notifications during:

| Context | Detection Method | Apps Detected |
|---------|------------------|---------------|
| **Gaming** | Process + window | Steam, Lutris, Wine games |
| **Videos** | Fullscreen + window | YouTube, Netflix, VLC, MPV |
| **Presenting** | Window titles | Impress, PowerPoint, screen share |
| **Meetings** | Process names | Zoom, Teams, Meet, Webex, Discord |

**Features**:
- 10-second poll interval
- Automatic enable/disable
- Background monitoring
- Zero manual intervention

**Usage**:
```bash
aether-smart-notifications.sh enable   # Enable feature
aether-smart-notifications.sh monitor  # Start monitoring
aether-smart-notifications.sh status   # Check status
```

## ðŸ›  Part 6: Utilities & User Comfort

### QuickPal Launcher
**Script**: `scripts/aether-quickpal.sh`

Spotlight-style quick access to:
- System tools (Control Center, Health Check, etc.)
- Performance modes (Focus Mode, CleanMode, etc.)
- System settings (Display, Network, Power, etc.)
- Common applications (File Manager, Terminal, etc.)

**Features**:
- Uses kdialog (KDE) or zenity (fallback)
- Keyboard shortcut support
- Categorized menu items
- Quick toggle for all v2.0 features

**Usage**:
```bash
aether-quickpal.sh            # Show launcher
aether-quickpal.sh search focus  # Search (future)
```

**Tip**: Bind to `Super+Space` for instant access!

### Profile Sync
**Script**: `scripts/aether-profile-sync.sh`

Save and restore all preferences in one click:

**What Gets Saved**:
- Theme and color scheme
- Wallpaper
- Performance profile
- Blur settings
- CleanMode status
- Focus Mode settings
- Smart notifications
- Sound theme
- Touchpad settings
- Animation speed

**Features**:
- JSON-based profiles
- Automatic backups with timestamps
- Multiple named profiles (work, gaming, personal)
- One-click restore

**Usage**:
```bash
aether-profile-sync.sh save              # Save default
aether-profile-sync.sh save work         # Save as "work"
aether-profile-sync.sh restore work      # Restore "work"
aether-profile-sync.sh list              # List profiles
```

**Files**:
- Profiles: `~/.aether-profile.json` (default)
- Named: `~/.aether-profile-{name}.json`
- Backups: `~/.aether-profile-backups/`
